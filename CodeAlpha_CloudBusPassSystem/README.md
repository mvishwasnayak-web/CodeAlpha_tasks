# Cloud-Based Bus Pass System

A web-based bus pass booking and management system developed using Flask.

## Features

- View Bus Routes
- Book Bus Passes
- Generate Unique Pass IDs
- Search Pass by ID
- View All Bookings
- Fixed Route Pricing
- Modern Responsive Interface

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Bootstrap
- JSON

## Scalability

The application can be deployed on cloud platforms such as Render, AWS, Azure, or Google Cloud. For large-scale deployments, JSON storage can be migrated to PostgreSQL or MySQL databases and multiple application instances can be provisioned behind a load balancer.

## Reliability Features

- Unique Pass ID generation
- Digital ticket storage
- Fixed route pricing
- Pass retrieval through search
- Centralized booking management
